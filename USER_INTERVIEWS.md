# User Interview 1

Role: Student developer working on AI projects

Key Feedback:

- Uses multiple AI tools simultaneously
- Never tracks monthly spending properly
- Interested in cheaper alternatives

Most surprising insight:
They underestimated their AI spending by almost 2x.

# User Interview 3

Role: Freelance developer

Key Feedback:
