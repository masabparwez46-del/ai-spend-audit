# Reflection

## Hardest Bug

The hardest issue was integrating Supabase correctly and handling async insert operations. Initially the loading state and success message were placed incorrectly inside the insert object which caused syntax issues.

I debugged this by carefully separating database logic from UI state updates.

---

## Decision I Reversed

Initially I wanted to add a much more advanced audit engine with multiple comparisons and AI-generated reports.

Midway through the project I realized that focusing on a simpler but polished MVP would be more realistic and reliable within the timeline.

This helped me prioritize usability and completion instead of unnecessary complexity.

---

## What I Would Build With More Time

If I had another week, I would:

- add proper authentication
- generate shareable audit pages
- add charts and analytics
- integrate real live pricing APIs
- build downloadable PDF reports
